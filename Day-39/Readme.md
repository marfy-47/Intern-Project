# Prescription Generator Web Application — 

### ✅ Prerequisites

- Java 21+
- Gradle 8+
- IntelliJ IDEA or any Java IDE
- Lombok Plugin enabled (if using Lombok)
- Spring Data JPA
- Spring Security with JWT Authentication and Authorization



## Project Structure

│ │ └── PrescriptionApplication.java
